Comparing The Ski Resort Superpasses (Python and SQL)

As an avid skier, I've always wished there was a way to easily compare ski resorts based on size, snowfall, and offerings. On top of that, I've been wondering recently about the aggregate differences between the multi-resort season passes offered by Vail Resorts (the Epic Pass) and Alterra Mountain Company (the Ikon Pass). In this project, I used Python to scrape for snowfall and other data about some of the resorts listed on Onthesnow.com, I explored and queuried the new database in Microsoft SQL Server, and finally I build a Tableau dashboard to visualize some of the data. I am currently working on a more informative writeup of my findings, but here you can find the code I used to scrape and queury the data. In the "Tableau Dashboards" section of this site you can find a dashboard I built to visualize some of this data.
